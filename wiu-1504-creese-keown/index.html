World Surf Project

About World Surf

World surf is site design to show host Usability test result for mobile and web page done by designers. Designers can use world surf to link to their wire frame test, so that user will be able to test their design to get feedback to where the user are looking and clicking on. World surf is to close the bridge between designer and user so the both parties can learn more about mobile and web application as the user tests the designs and the designer receive their results.
